# CSCI2020U-Assignment1
This is Assignment 1 for CSCI2020U. This was made with Gagandeep Pabla.

To download this assignment, click the Clone Or Download Button. Then click download as ZIP.

If you want to run the individual java files, through a terminal, navigate to <Question#>/src/<question#>.java. (To compile it in a terminal type javac <question#>.java, then java <question#>)
This was originally made with Netbeans IDE. So if you would like to compile it using netbeans, Select Open Project, And select the questions folder. Then on netbeans, navigate through the projects to find the .java file (same process as above), then click the Play/Run button at the top of the screen (the green arrow).
